# たんじょうび号

8月2日の3歳のお誕生日を楽しみに待つ、電車好きの子どものためのカウントダウンアプリです。

## Features

- 日本時間を基準に8月2日までの日数を自動計算
- 誕生年から年齢を自動計算し、翌年以降も正しく表示
- 日付をまたいだときや画面に戻ったときに自動更新
- 誕生日当日だけの特別なお祝い表示
- 次の誕生日にもそのまま使える年次切り替え
- 線路の進捗表示、汽笛、紙吹雪のインタラクション
- ボタンを押すと「きのう」から「きょう」へ残日数が1つ減るデイリー発車体験
- 緑の環状電車・赤い特急・黄色い検査電車による3種類の全画面アニメーション
- 「あと何駅」と誕生日駅を直接示す、幼児にも分かりやすい路線図
- 毎日絵柄が変わる「きょうの駅スタンプ」
- スマートフォン、タブレット、PC対応
- スマートフォン専用に再構成した縦長イラスト
- `prefers-reduced-motion` とキーボード操作に対応

## Local preview

静的サイトなので、任意のHTTPサーバーでルートディレクトリを公開してください。

```sh
python -m http.server 4173
```

`http://localhost:4173/?date=2026-08-02` のように `date` を付けると、表示確認用に日付を固定できます。

## Deployment

GitHub Actions が `main` ブランチへのpushを検知し、GitHub Pagesへ自動公開します。
