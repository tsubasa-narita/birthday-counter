# E235系 たんじょうび号

8月2日の誕生日まで、E235系1000番台をイメージした横須賀・総武快速線カラーの3D電車で毎日1駅ずつ進む、スマートフォン向けカウントダウンアプリです。

公開URL: https://tsubasa-narita.github.io/birthday-counter/

## あそびかた

1. 大きな「ゆっくり しゅっぱつ」ボタンを押します。
2. きのうの残り日数駅から、きょうの残り日数駅まで電車が走ります。
3. 到着後は「もういちど はしる！」で、同じ旅を何度でも再生できます。

残り日数は端末の時計から日本時間で計算するため、同じ日に何度再生しても余計に減りません。右上のボタンでオリジナル走行音を切り替えられます。

## 見どころ

- Three.jsによるリアルタイム3DのE235系風4・11・15両編成
- 銀色車体、青とクリームの帯、前面マスク、灯火、ドア、窓、台車、パンタグラフ、グリーン車をプロシージャル生成
- 各車両が個別にカーブへ追従する長編成と、発車・追走・駅接近をつなぐカメラワーク
- 海、起伏のある地形、沿岸の家並み、トンネル、架線、信号、誕生日駅を組み立てた3Dジオラマ
- 画像生成機能で制作した、このアプリ専用の横須賀沿岸風オリジナル背景
- Web Audio APIで合成したオリジナル走行音・到着音
- 日本時間、誕生日当日、翌年への切替、うるう年、数日ぶりの追いつき運転に対応
- 端末性能別の描画品質、縮小モーション、WebGL非対応フォールバック、キーボード操作に対応

実在車両の特徴を尊重した独自の3D表現で、鉄道会社のロゴ、写真、公式素材、実際の発車メロディーや走行音は使用していません。

## 開発

```sh
npm ci
npm run dev
npm test
npm run build
```

Viteのベースパスは `/birthday-counter/` に設定されています。確認用URLパラメータは次の通りです。

- `?date=2026-08-02`: 表示日を固定
- `?motion=full` / `?motion=reduce`: モーションモードを固定
- `?webgl=off`: 軽量フォールバックを確認
- `?scene=0.5`: 3Dカメラの進捗位置を固定

`main` へのpushでGitHub Actionsが依存関係の固定インストール、テスト済み本番ビルド、`dist`のGitHub Pages公開を行います。
