# たんじょうび号

8月2日の3歳のお誕生日を楽しみに待つ、電車好きの子どものためのカウントダウンアプリです。

公開URL: https://tsubasa-narita.github.io/birthday-counter/

## 使い方

1. 好きな電車を1つ選びます。
2. 大きな「おして しゅっぱつ！」ボタンを押します。
3. 電車が誕生日駅へ走り、残り日数が「きのう」から「きょう」へ1日進みます。
4. 到着後は「ホームに もどる」か「つぎの でんしゃでもういっかい」を選べます。

右上の「おと あり／おと なし」で音を切り替えられます。毎日初めて到着した日は端末に記録されますが、何度再生しても実際の残り日数が余計に減ることはありません。

## 主な機能

- 日本時間を基準に8月2日までの残り日数と年齢を自動計算
- 前回の8月2日から次回の8月2日までを正確に表す路線図
- オリジナルの緑の環状電車、赤い特急、黄色い検査電車
- 高解像度の生成画像を使った全画面走行・到着アニメーション
- 音のオン／オフ、途中終了、別車種での再実行
- スマートフォン縦横、セーフエリア、キーボード操作に対応
- `prefers-reduced-motion` でも意味が伝わる短縮演出
- JavaScriptや保存領域、音声が使えない場合のフォールバック

## ローカル確認

静的サイトなので、リポジトリ直下で任意のHTTPサーバーを起動します。

```sh
python -m http.server 8765
```

`http://localhost:8765/?date=2026-08-02` のように `date=YYYY-MM-DD` を付けると、境界日の表示を確認できます。テスト用に `motion=full` または `motion=reduce` も指定できます。

## 公開

`main` ブランチへのpushをGitHub Actionsが検知し、GitHub Pagesへ自動公開します。HTMLはバージョン付きCSS・JavaScriptを参照するため、スマートフォンに古いファイルが残った場合の組み合わせ不整合を防ぎます。

## 画像

ヒーロー、3種類の電車、走行背景は、このアプリ専用にOpenAIの画像生成機能で制作したオリジナル画像です。実在する鉄道会社のロゴ、車番、固有の車両デザインは使用していません。
